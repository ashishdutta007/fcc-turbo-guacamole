Request Header Parser Microservice API

This is a microservice API project for Free Code Camp that will return the ip address, language selected in the browser and type of os of the requestors device.

Usage:

https://immense-brushlands-3354.herokuapp.com/api/whoami/

Example Response:

			   {"ipaddress":"103.227.99.186","language":"en-US","software":"Windows NT 6.3; Win64; x64"}
